API Board Procedures¶

getBoard¶

  • Purpose: Get all necessary information to display a board
  • Parameters:
    • project_id (integer, required)
  • Result on success: board properties
  • Result on failure: empty list

Request example:

{
    "jsonrpc": "2.0",
    "method": "getBoard",
    "id": 827046470,
    "params": [
        1
    ]
}
