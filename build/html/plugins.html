Разработка плагина¶

Внимание: API плагинов на данный момент в состоянии альфа.

Плагины удобны для расширения базового функционала Канборда: добавление возможностей, создание тем или изменения базового поведения.

Создатели плагина должны указать точную версию Канборда, под которую написан плагин. Внутренний код Канборда может изменяться и ваш плагин должен тестироваться на совместимость с новой версией. Всегда следите за ChangeLog для внесения изменений.
