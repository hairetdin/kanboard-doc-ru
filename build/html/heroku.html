Развертывание Канборд на Heroku¶

Вы можете бесплатно испытать работу Kanboard на Heroku. Вам нужно нажать кнопку Deploy to Heroku и следовать руководству приведенному ниже:

Требования¶

  • Учетная запись на Heroku. Вы можете зарегистрироваться бесплатно.

  • Установленная утилита командной строки Heroku

Руководство по установке¶

# Get the last development version
git clone https://github.com/fguillot/kanboard.git
cd kanboard

# Push the code to Heroku (You can also use SSH if git over HTTP doesn't work)
heroku create
git push heroku master

# Start a new dyno with a Postgresql database
heroku ps:scale web=1
heroku addons:add heroku-postgresql:hobby-dev

# Open your browser
heroku open

Ограничения¶

  • Хранилище на Heroku эфимерное. Это означает, что файлы, загружаемые через Канборд, будут отсутствовать в системе после перезагрузки. Вы можете установить плагин для хранения файлов в облаке, например Amazon S3.

  • Некоторые возможности Канборда требуют, чтобы вы выполняли запуск ежедневных фоновых задач.
