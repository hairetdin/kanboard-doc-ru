Как создать asset (Javascript и CSS файлы)¶

Файлы CSS стилей и Javascript объединены вместе и минимизированы.

  • Оригинальные файлы CSS хранятся в каталоге assets/css/src/*.css

  • Оригинальные файлы Javascript хранятся в каталоге assets/js/src/*.js

  • assets/*/vendor.min.* - внешние зависимости объединены и минимизированы

  • assets/*/app.min.* - исходный код приложения объединены и минимизированы

Требования¶

  • NodeJS с npm

Сборка файлов Javascript и CSS¶

Канборд использует Gulp для сборки asset и Bower для управления зависимостями. Эти утилиты устанавлены в проекте как зависимости NodeJS.

Запустить все¶

make static

Собрать vendor.min.js и vendor.min.css¶

gulp vendor

Собрать app.min.js¶

gulp js

Собрать app.min.css¶

gulp css

Примечание¶

Сборка asset невозможна из архива Kanboard, вы должны клонировать репозиторий.
