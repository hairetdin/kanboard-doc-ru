API Project Procedures¶

createProject¶

  • Purpose: Create a new project
  • Parameters:
    • name (string, required)
    • description (string, optional)
  • Result on success: project_id
  • Result on failure: false

Request example:

{
    "jsonrpc": "2.0",
    "method": "createProject",
    "id": 1797076613,
    "params": {
        "name": "PHP client"
    }
}

Response example:

{
    "jsonrpc": "2.0",
    "id": 1797076613,
    "result": 2
}
