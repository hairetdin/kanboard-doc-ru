API Application Procedures¶

getVersion¶

  • Purpose: Get the application version
  • Parameters: none
  • Result: version (Example: 1.0.12, master)

Request example:

{
    "jsonrpc": "2.0",
    "method": "getVersion",
    "id": 1661138292
}

Response example:

{
    "jsonrpc": "2.0",
    "id": 1661138292,
    "result": "1.0.13"
}

getTimezone¶

  • Purpose: Get the application timezone
  • Parameters: none
  • Result on success: Timezone (Example: UTC, Europe/Paris)
  • Result on failure: Default timezone (UTC)

Request example:

{
    "jsonrpc": "2.0",
    "method": "getTimezone",
    "id": 1661138292
}

Response example:

{
    "jsonrpc": "2.0",
    "id": 1661138292,
    "result": "Europe\/Paris"
}
