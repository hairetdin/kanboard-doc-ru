Environment Variables
=====================

Environment variables maybe useful when Kanboard is deployed as
container (Docker).

+-----------------+-------------------------------------------------------------------------------------------------------------------------------------+
| Variable        | Description                                                                                                                         |
+=================+=====================================================================================================================================+
| DATABASE\_URL   | ``[database type]://[username]:[password]@[host]:[port]/[database name]``, example: ``postgres://foo:foo@myserver:5432/kanboard``   |
+-----------------+-------------------------------------------------------------------------------------------------------------------------------------+
| DEBUG           | Enable/Disable debug mode: "true" or "false"                                                                                        |
+-----------------+-------------------------------------------------------------------------------------------------------------------------------------+
| LOG\_DRIVER     | Logging driver: stdout, stderr, file or syslog                                                                                      |
+-----------------+-------------------------------------------------------------------------------------------------------------------------------------+
