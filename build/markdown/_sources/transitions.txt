Task transitions
================

Transitions record each movement of the tasks between columns.

.. figure:: https://kanboard.net/screenshots/documentation/transitions.png
   :alt: Transitions

   Transitions
Available from the task view, you can see that information:

-  Date of the action
-  Source column
-  Destination column
-  Executor (users that moves the task)
-  Time spent in the origin column

Task transition data can also be exported from the project settings
page.

.. figure:: https://kanboard.net/screenshots/documentation/transitions-export.png
   :alt: Transitions Export

   Transitions Export
For the specified time range you will generate a CSV file that you can
use with any spreadsheet software.
